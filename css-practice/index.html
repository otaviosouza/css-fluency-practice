positions

#01 Block - position: static

It is the default behavior and displacement properties as top, left and others do not affect it.

#02 Block - position: absolute

This block is child of #01 Block and has absolute position.
It has "0" value to properties bottom and right.

It is not following the HTML flow because it looks for the next non-static element. If it doesn't exist it takes the position from the viewport represented by the <html> element.

#04 Block - position: fixed

It takes the position from the viewport represented by the <html> element.

#05 Block - position: relative

It has the same behavior of static position because it does not have displacement properties.

#06 Block - position: relative

This block has displacement properties.

#03 Block - position: absolute

This block is child of #06 Block and has absolute position.
It has "0" value to properties bottom and right.

As the parent element is non-static element, it follows the HTML flow and doesn't take the position from the viewport.
